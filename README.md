# Practica02-ICC

## Objetivo
Creación de usuarios con nombre, correo y rol.

## Descripción:
Este proyecto busca la creación correcta de usuarios, ya sea empleados o administradores. Utilizando getters y setters, además de verificación de errores.

## Tecnologías utilizadas
Java

## Instrucciones de Ejecución
Compilar desde src: "javac modulos/Usuario.java Main.java"

Ejecutar el programa desde src: "java Main"
